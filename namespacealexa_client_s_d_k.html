Detailed Description

Whether or not curl logs should be emitted.

Handle HTTP2 request / response for a ping sent to AVS.

Deprecated:
This registration file is not needed if an application uses the manufactory to build its components. For example, the SDK Preview App does not use this registration file any longer.

However, for applications that have yet to transition to using the manufactory, this file is provided so those applications can continue to use Keyword Detection (for example, the backwards-compatible SDK Sample App does use this file).

Test of acsdkManufactory's compile time utilities.
